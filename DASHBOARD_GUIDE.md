## ğŸ” What Each Metric Means

### Network Metrics

- **Degree**: Number of direct connections a user has
- **PageRank**: Overall influence (like Google's algorithm)
- **Betweenness**: How often a user bridges communities
- **Closeness**: How central a user is in the network
- **Network Density**: How interconnected the network is (0-1)

### Content Metrics

- **Sentiment Score**: 0 (negative) to 1 (positive)
- **Viral Score**: Predicted engagement potential (0-1)
- **Importance**: Topic relevance score
- **Velocity**: Growth rate (recent vs historical mentions)

### Community Metrics

- **Union-Find**: Fast connected component detection
- **Greedy Modularity**: Optimized for community structure quality
- **Community Size**: Number of members in a group

## ğŸ› Troubleshooting

### Dashboard won't start

```bash
# Ensure Streamlit is installed
pip install streamlit --upgrade

# Check for port conflicts
streamlit run dashboard.py --server.port 8502
```

### No data showing

1. Verify analysis files exist in `output/` directory
2. Check file naming matches pattern: `{subreddit}_*.json/csv`
3. Re-run analysis if files are missing

### Visualization not loading

1. Check `{subreddit}_graph.html` exists
2. Verify file size (very large networks may be slow)
3. Try refreshing the page

### Performance issues

```bash

# Or sample larger datasets in dashboard.py (already implemented for graphs)
```
